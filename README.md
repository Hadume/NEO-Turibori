# NEO-Turibori  
# 概要  
「第53回Minecraft魅せるコマンド選手権」への応募作品の配布です！！  
  
# バージョン  
1.18.1  
# 遊び方  
- 釣りをしていると、上にバーが出てくるよ  
  そのバーの真ん中にある範囲に合わせて「スニーク」を押そう！  
  すると、魚さんが襲ってくるよ  
  そしてら、上のバーの矢印のタイミングで、真ん中に表示されてる操作(スニークorオフハンド)をしよう！  
- 餌を「オフハンド」に持つと高いレアリティの魚が釣れやすくなるよ！  
- 水に入るとスコアが下がっちゃうよ
